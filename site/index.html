    Introdução

    Esta documentação tem como propósito apresentar, de forma clara e detalhada, as principais funcionalidades e o fluxo de uso de cada tela do sistema SIGAFBA Treinamentos CRM que funciona em conjunto obrigatório com o sistema SIGAFBA Administrativo.

    O conteúdo aqui descrito visa orientar e facilitar o trabalho de todos os usuários envolvidos no processo comercial, oferecendo uma visão completa das etapas que compõem o ciclo de vendas, desde o recebimento dos leads, passando pelas fases de negociação e acompanhamento de eventos com o cliente, até a finalização e o envio de contratos, bem como a gestão de presença e demais operações relacionadas.

    O SIGAFBA Treinamentos CRM é um sistema altamente parametrizável, desenvolvido para atender às necessidades específicas durante todo o período de negociação, mantendo sempre registrado o histórico de vida do cliente, seus negócios anteriores, links de inscrição, arquivos anexados e acompanhamento pós-venda.

    Requisitos do sistema

    Para acessar o sistema, deve-se possuir um computador com acesso à internet e um navegador, ou para acesso ao aplicativo, deve-se possuir um dispositivo celular Android ou iOS (Não recomendamos a instalação das versões para MacBook e demais computadores macOS disponibilizadas pela própria Apple em sua loja).

    Observação: É necessário ter o módulo do SIGAFBA Administrativo para utilizar todos os módulos do sistema de forma integrada.

    SIGAFBA Administrativo

    O sistema SIGAFBA Treinamentos CRM mantém comunicação direta e contínua com o SIGAFBA Administrativo (para mais informações consulte aqui), garantindo a integração completa entre as áreas comercial e administrativa. Todas as informações referentes a contratos, vendas finalizadas, participantes e processos de prospecção são automaticamente enviadas para o módulo administrativo, assegurando atualização em tempo real e centralização dos dados.

    O sistema SIGAFBA Administrativo possui acesso exclusivo para o gerenciamento de usuários, permitindo o cadastro no SIGAFBA Treinamentos CRM, de acordo com as permissões e funções definidas pelos administradores.

    Como acessar?

    O sistema pode ser instalado através do aplicativo SIGAFBA disponibilizado tanto para Android quanto para iOS (Veja mais sobre os requisitos do sistema aqui) ou também acessando via navegador através do link https://sigafba.com.br/treinamentos/entrar.
