# Turma Item (/turma/item)

## Resumo
Detalhe de uma turma específica e seus itens associados.

## URL
`/turma/item`

## Principais Ações
- Visualizar/editar itens.
- Gerenciar participantes.
- Ajustar cronograma.

## Fluxo da Tela
1. Selecionar turma.
2. Operar sobre itens/participantes.
3. Salvar.

## Relacionados
- Turma (`/turma`)
- Turma Lista (`/turma/lista`)
