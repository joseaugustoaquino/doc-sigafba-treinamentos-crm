# Turma Link Registrar (/turma/link/registrar)

## Resumo
Criação/registro de novos links vinculados a turmas.

## URL
`/turma/link/registrar`

## Principais Ações
- Definir parâmetros do link.
- Salvar e disponibilizar.

## Fluxo da Tela
1. Preencher dados do link.
2. Registrar e validar.

## Relacionados
- Turma Link Lista (`/turma/link/lista`)
