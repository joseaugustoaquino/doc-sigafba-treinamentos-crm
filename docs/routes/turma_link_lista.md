# Turma Link Lista (/turma/link/lista)

## Resumo
Gestão e visualização de links relacionados a turmas.

## URL
`/turma/link/lista`

## Principais Ações
- Listar links.
- Ativar/desativar.
- Copiar/compartilhar.

## Fluxo da Tela
1. Acessar a lista de links.
2. Operar sobre cada link.

## Relacionados
