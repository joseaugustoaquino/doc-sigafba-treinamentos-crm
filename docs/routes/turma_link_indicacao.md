# Turma Link Indicação (/turma/link/indicacao/:contract/:environment/:idClass/:idUser/:idParticipant?)

## Resumo
Fluxo de indicação para turmas via link parametrizado.

## URL
`/turma/link/indicacao/:contract/:environment/:idClass/:idUser/:idParticipant?`

## Parâmetros
- `:contract`, `:environment`
- `:idClass`, `:idUser`
- `:idParticipant?` (opcional)

## Principais Ações
- Realizar indicação.
- Validar turma/usuário/participante.

## Fluxo da Tela
1. Acessar via link.
2. Validar parâmetros.
3. Confirmar indicação.

## Relacionados
- Turma (`/turma`)
- Turma Link Lista (`/turma/link/lista`)
